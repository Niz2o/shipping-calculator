# Shipping Calculator

هذا المشروع هو تطبيق ويب بسيط لحساب تكاليف الشحن وتقديم تقديرات التسليم.

## كيفية الاستخدام

1. افتح الملف `index.html` في متصفح الويب الخاص بك.
2. أدخل الوزن والمسافة وانقر على "Calculate" لحساب التكلفة.
